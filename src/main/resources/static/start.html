REST URL References

URL Output
/device Respons is a JSON with all known devices and last seen command.
I no devices is configured you will get an empty respons with just two brackets "[ ]"
/device/1 Respons is a JSON with details of device with ID 1.
If there is no device with that ID you will get a 404 error.
/device/1/lastcmd Returns a string with the last know command sent to device with id 1. Example "ON", "OFF", "DIM 150" etc
Commands that you can send is ON, OFF, DIM {dimlevel}, BELL, UP, DOWN, STOP and EXECUTE.
If there is no device with that ID you will get a 404 error.
/device/1/set?cmd=ON Returns a string with the return status of command sent to device. "SUCCESS" if the call succeded and "FAILED" if something whent wrong.
If there is no device with that ID you will get a 404 error.
/sensor Respons is a JSON with all known sensors seen the last 30 minutes.
If there is no sensors registered you will get an empty respons with just two brackets "[ ]"
/sensor/1 Respons is a JSON with details of thee sensor with ID 1.
If there is no sensor with that ID you will get a 404 error.
